🔗 Connect Discord (browser sign-in) One click — authorize, pick a server + channel, the webhook is captured automatically. First time only: add your Discord app's Client ID/Secret under Advanced.
Advanced / manual setup
  1. To enable Connect: discord.com/developers → New Application → OAuth2 → copy Client ID + Client Secret below; add redirect http://localhost:3737/auth/discord/callback.
  2. Or manual webhook: Discord channel → Edit Channel → Integrations → Webhooks → New Webhook → Copy URL → paste below.

Not configured

🔗 Connect Slack (browser sign-in) One click — authorize in your browser, no tokens to copy. First time only: add your Slack app's Client ID/Secret under Advanced.
Advanced / manual setup
  1. To enable Connect: api.slack.com/apps → your app → Basic Information → copy Client ID + Client Secret below.
  2. OAuth & Permissions → Redirect URLs → Add → paste http://localhost:3737/auth/slack/callback → Save.
